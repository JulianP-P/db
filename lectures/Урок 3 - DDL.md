## Основные объекты
### Кластер 
- База данных
- Табличное пространство
- Роль (пользователь)
### База данных
- Схема
  - Таблица
    - Индекс
  - Материализованное представление
  - Последовательность
  - Функция, процедура
  - Представление

Получение информации о бд (неважно, к какой бд подключились)
```sql
select * from pg_database
```
Подобная команда в psql
```
\l
\l+
```
